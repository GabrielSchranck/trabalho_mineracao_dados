Coleta, Tratamento e Análise Exploratória de Dados via Web Scraping

# Escolha do site

Para o trabalho de mineração de dados, foi escolhido o site da NBA. Essa escolha não foi por acaso, pois, sendo considerada a principal liga de basquete do mundo, possui uma grande variedade e quantidade de produtos.

Com a coleta de dados que será realizada, será possível fazer análises preditivas e identificar os produtos mais vendidos, que podem ser utilizados em estratégias de marketing, além de permitir a verificação dos principais interesses dos usuários.

Além disso, com essas informações sobre os produtos, também é possível realizar um processo de curadoria, organizando e estruturando os dados de forma adequada para comercialização com outras empresas que revendem os mesmos produtos.
